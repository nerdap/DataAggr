DataAggr v0.1 Alpha

Note:
Make sure you have the C++0x tr1 libraries with the compiler you're using.

Usage:
Start the application. Use Alt-Q to bring up the DataAggr window. Type something. Use Alt-Q again to save your text and hide the window again.

The default file to save notes in is notes.txt. If you want to save your notes in a different file, use the following syntax to enter the note:
:file(filename);Notes go here
